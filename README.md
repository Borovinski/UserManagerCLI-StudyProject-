# UserManagerCLI

Учебный проект на Go, демонстрирующий работу с пользователями через командную строку (CLI).

## Функционал

### Добавление пользователя
- Ввод имени, возраста и рейтинга.
- Проверка корректности введённых данных.

### Просмотр всех пользователей
- Вывод всех пользователей с полями: ID, имя, возраст, рейтинг.

### Поиск пользователей
- По ID
- По имени
- По возрасту
- По рейтингу

### Изменение данных пользователя
- Изменение ID
- Изменение имени
- Изменение возраста
- Изменение рейтинга

### Удаление пользователя
- Удаление пользователя по ID.

---

## Используемые технологии

- **Go** — основной язык разработки.
- **CLI** — взаимодействие с пользователем через командную строку.
- **Модули и пакеты Go** — для структурирования кода (`menu`, `storage`, `models`, `input`).

---

## Основные подходы

- Разделение **логики хранения данных** и **логики CLI**.
- Валидация данных в конструкторе и методах `User`.
- Проброс ошибок вверх к меню для корректного вывода пользователю.
- Использование мапы указателей для хранения пользователей (`map[int]*User`).
